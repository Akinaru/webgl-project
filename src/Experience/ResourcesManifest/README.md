# ResourcesManifest

Dossier reserve pour une evolution de la couche manifest.

## Etat actuel

Le projet utilise aujourd'hui `src/Experience/Source/` comme manifest central actif.

## Pourquoi ce dossier existe

Il peut servir plus tard si le manifest devient plus riche:

- separation par batches,
- metadonnees de post-traitement,
- schemas de validation,
- generation automatique.

## Regle actuelle

Tant que rien n'est migre ici, `Source/` reste la source de verite.
